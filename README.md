# git2codetour

これは、対象のGitローカルリポジトリに記録されているコミットログを解析して、指定された二つのコミット間の差分を CodeTour 形式で出力するコマンドラインツールです。

A command-line tool that analyzes commit logs recorded in a target Git local repository and outputs the differences between two specified commits in CodeTour format.

## インストール / Installation

```bash
npm install
npm run build
```

## 使い方 / Usage

```bash
node dist/index.js <from-commit> <to-commit> [options]
```

### Arguments

- `<from-commit>`: 開始コミットの参照 (SHA、ブランチ名、タグ名など) / Starting commit reference (SHA, branch name, tag name, etc.)
- `<to-commit>`: 終了コミットの参照 (SHA、ブランチ名、タグ名など) / Ending commit reference (SHA, branch name, tag name, etc.)

### Options

- `-r, --repo <path>`: Gitリポジトリのパス (デフォルト: カレントディレクトリ) / Path to git repository (default: current directory)
- `-o, --output <file>`: 出力ファイルのパス (デフォルト: 標準出力) / Output file path (default: stdout)
- `-V, --version`: バージョンを表示 / Display version
- `-h, --help`: ヘルプを表示 / Display help

## 使用例 / Examples

### 標準出力に出力 / Output to stdout

```bash
node dist/index.js abc123 def456
```

### ファイルに出力 / Output to file

```bash
node dist/index.js abc123 def456 -o changes.tour
```

### 別のリポジトリを指定 / Specify different repository

```bash
node dist/index.js abc123 def456 -r /path/to/repo
```

### ブランチ名を使用 / Use branch names

```bash
node dist/index.js main develop
```

## 出力形式 / Output Format

出力は CodeTour JSON 形式です。各ステップには以下の情報が含まれます:

The output is in CodeTour JSON format. Each step contains the following information:

```json
{
  "title": "Changes from abc123 to def456",
  "description": "Diff between commits...",
  "steps": [
    {
      "file": "path/to/file.js",
      "line": 42,
      "description": "**Added:**\n```\ncode here\n```"
    }
  ]
}
```

## CodeTourについて / About CodeTour

CodeTourは、コードベースのガイド付きウォークスルーを記録・再生できるVisual Studio Code拡張機能です。
このツールで生成されたツアーファイルは、VSCodeのCodeTour拡張機能で使用できます。

CodeTour is a Visual Studio Code extension that allows you to record and play back guided walkthroughs of your codebases.
The tour files generated by this tool can be used with the CodeTour extension in VS Code.

詳細: https://github.com/microsoft/codetour
